# Org Mode

## Overview
Use the org-* helper tools and the structure-preserving workflow in
`references/org-mode.md` when working on org-mode files.

## Workflow
- Read `references/org-mode.md` before editing large org files or todo.org
- Follow the org-outline → org-extract → org-add-heading workflow
- Preserve heading hierarchy and ordering rules
